EVENT NAME: BLIND CODING

Description:

  • Round 1:-

    1. A simple code with syntax error will be given on paper.
    2. Participants have to correct the errors on the paper and
      type the code with Monitor Switched Off.
    3. Fifteen minutes will be given to type and correct the code.
    4. Based on the results of the first round the participants
      will be selected to the next round.

  • Round 2:-

    1. Only problem statement will be provided.
    2. Participants need to type the code in turbo c
      with monitor switched off.
    3. Twenty minutes will be given to each participant.
    4. Winners will be announced based on the results of
      compilation and execution.

Rules and regulations :

  1. Individual event.
  2. The compiler provided will be TURBO C.
  3. Consists of two rounds.
  4. The judges decision will be final.
